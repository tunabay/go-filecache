go-filecache example

This is an example image server. /image.png generates and caches images.

Examples:

  • /image.png (default parameters)
  • /image.png?color=cc0000 (change the color)
  • /image.png?width=800&height=600&color=0000cc&twist=-30 (more parameters)
  • /image.png?color=6666cc&twist=720
  • /image.png?color=ff0000&twist=-720&stripe=16
  • /image.png?color=ffff00&twist=0&stripe=64
  • /image.png?color=990099&twist=480&stripe=48&circle-width=1
  • /image.png?color=ff6600&twist=-60&stripe=3&circle-width=5
  • /image.png?color=ff6699&twist=30&stripe=48&circle-width=1

Parameters:

width
Image width in pixel. [16..]
height
Image height in pixel. [16..]
color
Key color in RRGGBB hex format.
twist
Amount to twist in degrees, can be negative.
circle-width
Line thickness of the concentric circles. [1..16]
stripe
Number of radiation stripes. [1..90]
